Install Git - Download from Internet

Check correct installation
Issue this command in the command Prompt
git --version

Open Command Prompt

git config --global user.name xyzxyzyxz
git config --global user.email xyzxyzxyz@gmail.com


Open Git bash

Generate the key (use your email):
ssh-keygen -t rsa -b 4096 -C "xyzxyzxyz@gmail.com"

When it asks:
Enter file in which to save the key (/c/Users/Paul/.ssh/id_rsa):

press Enter.

Enter passphrase (empty for no passphrase):

For a test project, just press Enter twice (leave it empty). If you want extra security, you can set one.

Start the SSH agent:
eval "$(ssh-agent -s)"

Add the key:
ssh-add ~/.ssh/id_rsa
Copy the public key:
clip < ~/.ssh/id_rsa.pub

Then go to GitHub → Settings → SSH and GPG keys → New SSH key, paste it, and save.

Test it:
ssh -T git@github.com

The first time, type:
yes

You should then see:
Hi xyzxyzxyz! You've successfully authenticated, but GitHub does not provide shell access.
