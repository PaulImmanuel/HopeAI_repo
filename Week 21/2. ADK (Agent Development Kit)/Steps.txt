1. Create root folder

D:\Hope_AI_GitHub_version\HopeAI_repo\Week 21\2. ADK (Agent Development Kit)

--------------------------------------------------

2. Open VS Code in:

D:\Hope_AI_GitHub_version\HopeAI_repo\Week 21\2. ADK (Agent Development Kit)

--------------------------------------------------

3. Open terminal in VS Code

--------------------------------------------------

4. Create virtual environment

py -3.12 -m venv venv

--------------------------------------------------

5. Activate virtual environment

.\venv\Scripts\Activate.ps1

if error 
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

--------------------------------------------------

6. Install packages

pip install google-adk==1.32.0 litellm==1.83.14 openai==2.24.0 python-dotenv==1.2.2                    

--------------------------------------------------

7. Create folder structure

D:\Hope_AI_GitHub_version\HopeAI_repo\Week 21\2. ADK (Agent Development Kit)│
├── agents
│   │
│   ├── math_agent
│       │
│       ├── __init__.py
│       ├── .env
│       ├── agent.py
│
├── venv

--------------------------------------------------

8. Add API key inside:

agents/math_agent/.env

Content:

GOOGLE_API_KEY=YOUR_API_KEY

(No quotes)

--------------------------------------------------

9. Add ADK agent code inside:

agents/math_agent/agent.py

--------------------------------------------------

10. Ensure terminal is inside:

D:\Hope_AI_GitHub_version\HopeAI_repo\Week 21\2. ADK (Agent Development Kit)

--------------------------------------------------

11. Run ADK

adk web agents

--------------------------------------------------

12. Open browser

http://127.0.0.1:8000

--------------------------------------------------

13. Select:

math_agent

from dropdown.

--------------------------------------------------

14. Test agent in chat UI.
